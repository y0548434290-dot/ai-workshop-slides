מושג רקע
מה זה Claude Code בדפדפן?
סוכן AI שמריץ את המשימות על תשתית ענן מנוהלת של Anthropic.
claude.ai/code
  • כל משימה רצה ב-sandbox (סביבה מבודדת ובטוחה) משלה בענן.
  • מתחברים פעם אחת ל-GitHub, ו-Claude מושך את הקוד, עובד עליו, ומחזיר תוצאה.
  • אפס התקנה — לא צריך VS Code, Node או טרמינל על המחשב.
ה-session ממשיך לרוץ גם אם סוגרים את הדפדפן — אפשר לחזור אליו מתי שרוצים.

1כניסה והתחברות
נכנסים לכתובת ומתחברים
  1. פותחים בדפדפן את הכתובת:
claude.ai/code
  1. מתחברים עם חשבון ה-Claude שלך — אותו חשבון של המנוי (Pro/Max/Team).
זה עובד גם מהדפדפן בטלפון — אפשר להתחיל משימה מכל מקום.
2חיבור GitHub
מחברים את חשבון ה-GitHub — פעם אחת
  1. לוחצים לחבר את GitHub ומאשרים את ההרשאה (OAuth).
  2. בוחרים לאילו ריפו (מאגרי קוד) לתת גישה.
מאובטח: ה-token של GitHub לא נשמר בתוך ה-sandbox. Claude ניגש דרך proxy מאובטח, כך שגם אם משהו ישתבש — החשבון שלך מוגן.
3משימה ראשונה
בוחרים ריפו, מתארים, ו-Claude יוצא לדרך
  1. בוחרים את הריפו שעליו רוצים לעבוד.
  2. כותבים בשפה חופשית מה רוצים שיקרה.
  3. Claude פותח session בענן ומתחיל לעבוד.
"תוסיפי דף "צור קשר" עם טופס, ותתקני את התצוגה במובייל"
אפשר לפתוח כמה משימות במקביל — כל אחת ב-session נפרד משלה.

מעבר לטרמינל

אפשר להעביר session מהדפדפן למחשב שלך ולהמשיך מקומית:

claude --teleport
מתחילים מהר בענן, וכשרוצים שליטה מלאה — ממשיכים מקומית. בלי להתחיל מחדש.
