התקנת VS Codeעורך הקוד המרכזי
חינם

VS Code הוא כלי העבודה המרכזי של מפתחים כיום. הוא קל, מהיר ותומך בכל סוגי הקוד.

  1. היכנסי לאתר הרשמי: code.visualstudio.com
  2. לחצי על כפתור Download המתאים למערכת ההפעלה שלך (Mac / Windows).
  3. הריצי את קובץ ההתקנה ועקבי אחר ההוראות על המסך.
אם את ב-Windows, בשלב ההתקנה סמני את האפשרות "Add to PATH" — זה יחסוך לך בעיות בהמשך.
2
התקנת Gitמערכת ניהול גרסאות
חינם

Git הכרחי כדי שנוכל לשמור, לנהל ולהעלות את הקוד לאוויר. גם Claude Code משתמש בו כדי להבין את מבנה הפרויקט.

  1. היכנסי לאתר: git-scm.com/downloads
  2. הורידי את הגרסה המתאימה למערכת שלך.
  3. הריצי את קובץ ההתקנה — השאירי את כל ברירות המחדל (Next, Next, Next) עד לסיום.
  4. פתחי טרמינל (שורת פקודה) ובדקי שההתקנה הצליחה:
git --version

אם מופיע מספר גרסה — הכל תקין.

הגדרת הזהות שלך ב-Git

המערכת צריכה לדעת מי "חותמת" על הקוד. פתחי CMD או PowerShell (חפשי "PowerShell" בתפריט התחל) והריצי את שתי הפקודות — אחת אחרי השנייה:

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
חשוב מאוד: השתמשי בדיוק באותה כתובת מייל שאיתה תפתחי חשבון GitHub בשלב הבא — כך המערכות יזהו שזו את ויסתנכרנו.
3
התקנת Node.jsדרישת קדם להרצה
חינם

כדי להריץ את ה-CLI של Claude Code, אנחנו צריכות את סביבת Node.js (גרסה 18 ומעלה).

  1. היכנסי לאתר: nodejs.org
  2. הורידי את גרסת ה-LTS (כתוב עליה "Recommended for most users").
  3. הריצי את קובץ ההתקנה לפי ברירות המחדל.
  4. בדקי בטרמינל שההתקנה הצליחה:
node --version
הפקודה אמורה להציג מספר שמתחיל ב-v18 ומעלה, למשל: v20.11.0
4
פתיחת חשבון GitHubהענן של המפתחים
חינם

GitHub הוא ה"ענן" שאליו נעלה את הקוד באמצעות Git — כדי לגבות אותו, לשתף אותו, ולחבר אותו לשירותי אחסון כדי שהאתר או המערכת יהיו באוויר.

  1. היכנסי לאתר: github.com
  2. לחצי על Sign up בפינה הימנית העליונה.
  3. הזיני את אותה כתובת מייל שהגדרת ב-Git (שלב 2), בחרי סיסמה חזקה ושם משתמש פנוי.
  4. עקבי אחר הוראות האימות ואשרי את המייל שתקבלי לתיבת הדואר.
5
התקנת Claude Codeסוכן ה-AI שכותב איתנו קוד
דורש מנוי

יש שתי דרכים עיקריות לעבוד עם Claude Code. בשיעור נתמקד בשתיהן:

מסלול א' — תוסף ב-VS Codeמומלץ למתחילות

הדרך הנוחה ביותר — הכל מתוך עורך הקוד.

  1. פתחי את VS Code.
  2. לחצי על אייקון ה-Extensions (ריבוע בצד שמאל) או הקישי Ctrl+Shift+X.
  3. חפשי את התוסף הרשמי: Claude Code (של Anthropic) ולחצי Install.
  4. אייקון כוכב (Spark) יופיע בסרגל הצד — סימן שהכל מוכן.

מסלול ב' — התקנה דרך הטרמינל (CLI)

למי שרוצה כוח מלא בטרמינל. פתחי טרמינל ב-VS Code (Ctrl+`) והריצי:

npm install -g @anthropic-ai/claude-code
Claude Code דורש מנוי Claude Pro — מינימום $20 לחודש. ניתן להתקין ולהתחבר ישירות מתוך VS Code.
תוסף מומלץ לנוחות בעברית: Claude Code RTL — מייעל את הממשק לכתיבה מימין לשמאל.

התחברות ואימות (Login)השלב האחרון — את כמעט שם!

כדי ש-Claude יוכל לעבוד, את צריכה להתחבר לחשבון שלך:

  1. בטרמינל, הקלידי את המילה:
claude
  1. יופיע קישור בדפדפן — התחברי עם חשבון Claude Pro או ה-API Key שלך.
  2. כשהטרמינל מציג Ready — סיימת את ההתקנה ואת מוכנה לפתח! 🎉
